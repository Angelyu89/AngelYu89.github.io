	while循环与for循环的区别：
	while循环是未知的，且当条件不满足将停止执行。
	for循环是已知的也可以是未知的，当循环完成后后继续判断一直到结果为false;

	break
	
	var iNum=0;
	for(var i=1;i<10;i++){
		if(i%5==0){
			break;
		}
		
		iNum++;
	}
    alert(iNum);           
	在以上代码中，for 循环从 1 到 10 迭代变量 i。在循环主体中，if 语句将（使用取模运算符）检查 i 的值是否能被 5 整除。
	如果能被 5 整除，将执行 break 语句。alert 显示 "4"，即退出循环前执行循环的次数
	
	
	continue

			continue;
		}
		
		iNum++;
	}

	这里，alert 将显示 "8"，即执行循环的次数。
	可能执行的循环总数为 9，不过当 i 的值为 5 时，将执行 continue 语句，会使循环跳过表达式 iNum++，返回循环开头。
