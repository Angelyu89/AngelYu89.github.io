JS的引入方式 >

JS代码写在哪儿？

方法一

写到body尾部 最后一行 内部写法(一般不这么写)


script type="text/javascript"
console.log("hello world");
/script

写到body尾部 最后一行 引用外部JS 外部(页面加载完成才加载)


script src="main.js">/script

方法二

在head标记里面，通过body.onload事件执行( head中引入访问不到body的标记 eg: document.body.inHTML="hello world" )


script src="main.js">/script
function init(){ document.body.innerHTML="Hello world"; } body onload="init()"

方法三

写在head 标签里面，通过window.onload事件执行

script src="main.js">/script
window.onload=function(){ document.body.innerHTML="hello world"; }
